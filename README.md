## ESP32使用时注意
- 启动设置 芯片默认(2,12)内部下拉(0,5,15)内部上拉
- CMD CLK SD0 SD1 SD2 SD3 用于连接模块flash
- ADC1:32,33,(34,35,36,37,38,39)仅输入
- ADC2:0,2,4,12,13,14,15,25,26,27 打开WIFI不可用
